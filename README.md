# LangChain + LangGraph Demo: 검사지 자동화

이 프로젝트는 **LangChain**과 **LangGraph**를 활용하여  
CSV 형식의 간단한 검사지 데이터를 불러와, 학생별 **자동 요약 보고서**를 생성하는 Demo입니다.  

- 🧠 **입력**: 학생 검사 결과 (CSV)  
- ⚙️ **처리**: LangChain Prompt + LangGraph Workflow  
- 📄 **출력**: 학생별 맞춤형 요약 보고서
